pgSQLa.psqlAide.psqlAideCustom

The psqlAide function is a factory function that creates an instance of the psql generation system. It takes a ZodRawShape, a resolve function that uses the Injectables and a template function to generate psql text and an optional paOptions argument that provides a custom literal supplier for the resolve function's template function.

pgSQLa.psqlAide.FormatArgument

The FormatArgument type represents a named argument that should be included in a SQL format string. It includes a name, a type (represented by a Zod type), an index (indicating the argument's position), and three methods (s, L, I) that return the format string for the argument in different PostgreSQL format modes: normal, literal, and identifier.

pgSQLa.psqlAide.InjectableExpr

The InjectableExpr type represents an expression that can be included in psql text. It can be a Setable, a string, a number, a bigint, or a function that takes an index and an array of expressions and returns a Setable or primitive.
